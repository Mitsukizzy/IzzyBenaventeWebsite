Whiteboard Learning
A platform for classroom communities
Project Description
Whiteboard Learning is a platform for classroom communities that utilizes both mobile devices and All in One PCs. We developed our application for the AT&T IOT Hackathon where we won 1st place in the Education category and 3rd place in the overall competition.

I created the initial concept of the app and programmed the front end for web with mobile and All in One PCs in mind.
