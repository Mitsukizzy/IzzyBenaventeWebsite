ElemenTerra
VR Worldbuilding game
Project Description
ElemenTerra was one of five Advanced Game Projects developed at USC during the 2014-2015 school year. It is a virtual reality world-building game using the Oculus Rift and Sixense STEM. The team leads, Max and Camille, went on to create a company called Freeform Labs. In March 2016, ElemenTerra became a part of by Starbreeze Studios and they're doing some exciting work together. Contribution: Gameplay Engineer
While the game was an AGP, I developed an intuitive movement system using the Oculus Rift and Razer Hydra. I also programmed radial gravity on the planet and planting/growing on any surface. My favorite task was figuring out the best way to control flight and movement using both the Oculus and the Hydra so that the player did not experience any motion sickness. It was a great user experience challenge and valuable learning experience for me.
